Diffarenc Between LocalStorage And SessionStorage?

Local Stroage and Session Storage executing almost same.but they have some difference.

Local Storage

* Local Storage storage capacity is 5/10mb.

* Local Storage store untill deleted via JavaScript or manually.

* The client can only read local storage.

* There is no transfer of data to the server.

Session Storage

* Session storage storage capacity is 5mb.

* Session storage store data only for a session.that means if browser or tab is closed the session data is cleared.

* There is no transfer of data to the server.
